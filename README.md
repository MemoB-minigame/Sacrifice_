### 特效

全屏环绕特效：

依赖BuffManager、WeaponManager和Gun，读入状态并生成3种全屏环绕特效（无BUFF则不产生特效）。

由于实时渲染开销比较大，调试其它模块时请关闭全屏环绕特效：

```bash	
禁用./MemoB_Minigame_/Assets/URP/2D Renderer Data的名为'Blit'的Renderer Feature
禁用Player预制体的Scene Effect (Script)
```
